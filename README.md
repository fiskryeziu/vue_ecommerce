# Vue.js ecommerce App

A simple ecommerce application built with Vue.js. This is my first project using Vue.js, developed only by referring to the official Vue.js documentation.

## Features
- Product listing
- Shopping cart functionality
- Basic routing with Vue Router
- State management with Vue's built-in reactivity

## Tech Stack
- Vue.js
- Vue Router
- API for dummy data
- Optional: Pinia/Vuex for state management
- Optional: express/js with postgresql.

## Installation
```sh
# Clone the repository
git clone https://github.com/fiskryeziu/vue-ecommerce.git
cd vue-ecommerce

# Install dependencies
npm install

# Start the development server
npm run dev
```

## Usage
- Browse products
- Add/remove items from the cart
- Proceed to checkout (if implemented)
